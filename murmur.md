## 🛠️ Windows 已知問題與修補（2026-02-12 更新）

### ✅ 已修復問題

#### `spawn EINVAL` (安裝 Plugin 失敗)

- **狀態**: ✅ **已在 v2026.2.5+ 修復**
- **舊解法**: 手動修改 `exec-BIMFe4XS.js`
- **新解法**: 升級到最新版即可


### ⚠️ 持續追蹤問題

#### 本地模型 Memory 功能不穩定

- **現象**: Ollama 本地模型記憶功能時好時壞
- **官方狀態**: 已知 Bug (\#8470 / \#8131)
- **臨時對策**:

1. 升級 Ollama 至 **v0.15.5+**
2. 升級 OpenClaw 至 **v2026.2.8+**
3. 安裝 `supermemory` plugin
4. 等待 2 月底大更新


#### `unknown integration: openclaw`

- **狀態**: ✅ **已修復**
- **解決方案**: 升級 Ollama 至 v0.15.4+


### 📊 Windows 平台穩定度

| 功能 | Windows 原生 | WSL2 | 建議 |
| :-- | :-- | :-- | :-- |
| **基本對話** | ✅ 穩定 | ✅ 穩定 | 都可以 |
| **Memory** | ⚠️ 不穩定 | ✅ 穩定 | **用 WSL2** |
| **Skills** | ⚠️ 部分相容 | ✅ 完全相容 | **用 WSL2** |
| **Gateway** | ✅ 穩定 | ✅ 穩定 | 都可以 |
| **安全掃描** | ✅ 正常 | ✅ 正常 | 都可以 |

**結論**: 如果你需要完整功能，**強烈建議使用 WSL2**。
